# YogaUI

Design Files: https://www.uplabs.com/posts/yoga-class-mobile-app-dark-version-xd

## Using

* [Xamarin.Forms 4.7 Nightly](https://github.com/xamarin/Xamarin.Forms/wiki/Nightly-Builds)
* [LiveSharp](https://www.livesharp.net/)
